# Десктопный чат

### Основной функционал приложения: 
- Отправка сообщений 
- Удаление сообщений
- Поиск сообщений
- Выбор чата
- Поиск контактов
- Профиль активного чата
 ### Что я использовал: 
 - redux / redux-thunk;
 - PropTypes
 - REST API, асинхронные запросы на сервер
 - react-router-dom, classnames, dayjs, Css Transition
 - Git
